教育部先進資通安全實務人才培育計畫
校園資安深耕營Advanced Happy CyberSecurity Day

課程名稱：基礎現代密碼學

課程大綱：本課程介紹密碼學基本觀念與常用工具，包含古典密碼學、
亂數與雜湊函式及現代密碼的對稱式加密、

並針對RSA演算法攻擊進行說明，
